Teaching Plan

教學計劃書

此頁提供可直接閱讀的教學計劃。PPT 以網頁卡片方式瀏覽，不提供檔案下載。

課程目標

學生能理解微型餐飲創業如何從故事、菜單、空間、社群與會員制度整合成可執行方案。

核心案例

La Neige 法式雪冰室：台灣冰品、法式美感、父子退休創業、阿山哥俱樂部。

學習任務

完成一份品牌故事、一張菜單結構、一個社群月曆、一個會員活動與一個可展示網站原型。

成果評量

以品牌清楚度、商品可行性、互動設計、財務邏輯與網站展示完整度進行評分。

完整文字版計劃書：閱讀網頁版計劃書。範例網站：La Neige 法式雪冰室。

PPT Viewer

PPT 網頁瀏覽

以下為簡報重點瀏覽版，不提供 PPTX 檔案下載連結。

1. 品牌不是從 LOGO 開始

品牌從人、記憶與生活方式開始。琰綱與父親的故事，是法式雪冰室最重要的起點。

2. 市場定位

客單 90-150 元，以年輕拍照族、親子、退休族、山友與重機騎士作為主要社群。

3. 商品架構

夏季冰品、冬季熱飲湯品、會員活動互補，讓小店四季都有回訪理由。

4. 阿山哥俱樂部

PO 爬山照或重機照與品牌介紹，即可取得優惠碼，將生活熱情轉換為會員黏著。

5. GitHub Pages 展示

網站範例可作為課堂展示、作業參考與創業方案分享連結。
